# ☕ Bright Coffee Shop Sales Analysis

## Project Overview

This project analyzes transactional sales data from a coffee shop to provide insights into revenue, product performance, customer purchasing behavior, and store performance.

## Business Problem

The CEO wants to understand:

- Total revenue
- Best-selling products
- Peak sales periods
- Store performance
- Product category performance
